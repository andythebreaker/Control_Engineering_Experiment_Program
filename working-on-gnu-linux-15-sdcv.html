[完全用 GNU/Linux 工作] 15. 文字界面的字典 - sdcv

sdcv 可以說是文字界面的星際譯王 (StarDict)，它能使用星際譯王的字典詞庫，所以不用怕哪天沒人維護 StarDict，與其開啟圖形介面來查個單字，不如直接開終端機來的迅速。以下凍仁就列出幾項優缺點：
  • 優點：
    • 可於各種 terminator 使用，搭配 Guake 有加分效果。
    • 可使用相同的字典詞庫。
    • 可與純文字的工具搭配使用。
  • 缺點：
    • 需自備字典詞庫
    • 無法選字取詞。
    • 無法使用網路辭典。

1. 安裝


1.1. 安裝 sdcv。
[ jonny@wheezy ~ ]
# Debian, Ubuntu.
$ sudo aptitude install sdcv Enter

# CentOS, RHEL, Fedora (need EPEL repo)。
$ sudo yum install sdcv Enter

1.2. 下載字典檔 (星際譯王詞庫 | zh_TW)。
[ jonny@wheezy ~ ]
# 朗道英漢字典
$ wget http://abloz.com/huzheng/stardict-dic/zh_TW/stardict-langdao-ec-big5-2.4.2.tar.bz2 Enter

# 朗道漢英字典
$ wget http://abloz.com/huzheng/stardict-dic/zh_TW/stardict-langdao-ce-big5-2.4.2.tar.bz2 Enter

1.3. 於家目錄底下建立字典檔目錄。
[ jonny@wheezy ~ ]
$ mkdir -p $HOME/.stardict/dic Enter

1.4. 將字典檔解壓縮至 $HOME/.stardict/dic。
[ jonny@wheezy ~ ]
# 朗道英漢字典
$ tar jxvf stardict-langdao-ce-big5-2.4.2.tar.bz2 -C $HOME/.stardict/dic Enter

# 朗道漢英字典
$ tar jxvf stardict-langdao-ec-big5-2.4.2.tar.bz2 -C $HOME/.stardict/dic Enter

2. 基本使用


2.1. 列出有效的字典。
[ jonny@wheezy ~ ]
$ sdcv -l Enter
Dictionary's name Word count
朗道漢英字典5.0 395426
朗道英漢字典5.0 423246

2.2. 單字查詢 (只查一個單字)。
[ jonny@wheezy ~ ]
$ sdcv hello Enter
Found 1 items, similar to hello.
-->朗道英漢字典5.0
-->hello

*[hә'lәu]
interj. 喂, 嘿

2.3. 多重查詢 (進入無限查詢狀態)，使用 Ctrl + C 或 D 離開。
[ jonny@wheezy ~ ]
$ sdcv Enter
Enter word or phrase: 企鵝 Enter
Found 1 items, similar to 企鵝.
-->朗道漢英字典5.0
-->企鵝

penguin

$ cat $HOME/.sdcv_history | tail Enter

3. 補充說明


若想於終端機查 Yahoo! 奇摩字典，可改用 ydict [GitHub]。


▲ 豐有色彩的 ydict。

3.1. 下載 ydict。
[ jonny@wheezy ~ ]
$ wget http://ydict.googlecode.com/files/ydict-1.3.1.tar.gz Enter

3.2. 解壓縮。
[ jonny@wheezy ~ ]
$ tar zxvf ydict-1.3.1.tar.gz Enter
./._ydict
ydict

3.3. 查詢線上單字。
[ jonny@wheezy ~ ]
$ ./ydict penguin Enter
